# Global Cost of a Healthy Diet Dashboard

An interactive R Shiny dashboard exploring the cost of a healthy diet across countries and regions worldwide, measured in Purchase Power Parity (PPP) normalized to USD.

## About

This dashboard allows users to explore how the cost of a healthy diet varies across regions and countries between 2017 and 2024. Users can filter by region to compare costs, identify the top 5 countries with the highest cost increases over time, and explore the distribution of costs across regions.


## How to Install and Run

### Prerequisites
- R (version 4.0 or higher)
- RStudio

### Install required packages
```r
install.packages(c("shiny", "bslib", "tidyverse", "plotly"))
```

### Clone the repository
```bash
git clone https://github.com/luisalonso8/R-Shiny-App-Healthy-Diet-Cost.git
cd R-Shiny-App-Healthy-Diet-Cost
```

### Run the app
```r
shiny::runApp()
```
